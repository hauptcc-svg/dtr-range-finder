
### **Dashboard won't load**
```
Check: Flask is running
Refresh browser (Ctrl+Shift+R)
Check port 5000 is available
```

### **No auto-execution**
```
Check: Mode is "dtr" or "claude" (not "halted")
Check: Auto-execution shows "ON"
Wait 30 seconds for next check cycle
```

### **Missing files error**
```
Check: All 17 Python files copied
Check: templates/ folder exists with HTML
Check: public/ folder exists with JSON + JS
Check: .replit file in root
```
